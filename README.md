# 🧭 auto-doc-demo_mac_win

基于 **飞书多维表格 + Python + Sphinx** 的企业级自动化文档构建工具。

## 🚀 功能亮点
- 支持 CSV → RST → HTML/PDF 一键生成
- 模板继承与覆盖机制（类似 Espressif esp-docs）
- 支持 Mac/Windows 双端构建
- 飞书多维表格导出兼容
- 可扩展至多产品线（A1、A2...）

## 📘 使用示例
```bash
git clone https://github.com/Bingboom/auto-doc-demo_mac_win.git
cd auto-doc-demo_mac_win
make html
make pdf
