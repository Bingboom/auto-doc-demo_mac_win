AT+CSQ：获取信号强度
--------------

查询接收信号强度<rssi>。
命令格式

命令格式
^^^^^^^^



**执行命令**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**命令：**

::

    AT+CSQ

**响应：**

::

    <CR><LF>OK<CR><LF>




参数
^^^^


- **<service>**： 短信服务模式选择


  - 0：GSM03.40/GSM03.41 Phase 2

  - 1：GSM03.40/GSM03.41 Phase 2+



- **<mt>,<mo>,<bm>**： 下行/上行/广播支持


  - 0：不支持

  - 1：支持
