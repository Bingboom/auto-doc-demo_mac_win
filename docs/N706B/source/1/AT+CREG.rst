AT+CREG：查询网络注册状态
-----------------

查询模组的当前网络注册状态。
命令格式

命令格式
^^^^^^^^




**执行命令**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**命令：**

::

    AT+CREG

**响应：**

::

    <CR><LF>OK<CR><LF>





参数
^^^^


- **<service>**：
  短信服务模式选择


- **<mt>,<mo>,<bm>**：
  下行/上行/广播支持
