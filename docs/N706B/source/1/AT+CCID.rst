AT+CCID：获取SIM卡标识
-----------------

获取SIM卡的ICCID。
命令格式

命令格式
^^^^^^^^




**执行命令**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**命令：**

::

    AT+CCID

**响应：**

::

    <CR><LF>OK<CR><LF>





参数
^^^^


- **<service>**：
  短信服务模式选择


- **<mt>,<mo>,<bm>**：
  下行/上行/广播支持
