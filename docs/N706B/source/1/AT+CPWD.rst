AT+CPWD：修改密码指令
---------------

修改模组锁功能的密码。
若需修改PIN码,需锁定SIM卡（AT+CLCK="SC",1,"1234"）后才能修改。
命令格式

命令格式
^^^^^^^^




**执行命令**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**命令：**

::

    AT+CPWD

**响应：**

::


    <CR><LF>OK<CR><LF>





参数
^^^^


- **<service>**：
  
    短信服务模式选择


- **<mt>,<mo>,<bm>**：
  
    下行/上行/广播支持
