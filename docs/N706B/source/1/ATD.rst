.. _cmd-atd:

ATD：*99#：GPRS
--------------

使用外部协议栈,进行GPRS拨号连接。
进行拨号之前一定要确保CREG已经注册成功,并且设置了APN。
命令格式

命令格式
^^^^^^^^




**执行命令**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**命令：**

::

    ATD

**响应：**

::


    <CR><LF>OK<CR><LF>





参数
^^^^


- **<service>**：
  
    短信服务模式选择


- **<mt>,<mo>,<bm>**：
  
    下行/上行/广播支持
