AT+CLCK：PIN使能与查询功能指令
---------------------

锁、解锁以及查询MT和网络设备。设置该参数,重启模组后生效。
命令格式

命令格式
^^^^^^^^




**执行命令**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**命令：**

::

    AT+CLCK

**响应：**

::

    <CR><LF>OK<CR><LF>





参数
^^^^


- **<service>**：
  短信服务模式选择


- **<mt>,<mo>,<bm>**：
  下行/上行/广播支持
