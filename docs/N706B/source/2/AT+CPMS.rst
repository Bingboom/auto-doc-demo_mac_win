^^^^^^^^




**执行命令**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**命令：**

::

    AT+CPMS

**响应：**

::

    <CR><LF>OK<CR><LF>





参数
^^^^

  短信服务模式选择

  下行/上行/广播支持
