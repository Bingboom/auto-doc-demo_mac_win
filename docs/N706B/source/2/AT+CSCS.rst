AT+CSCS：设置TE字符集

设置TE字符集格式。

^^^^^^^^




**执行命令**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**命令：**

::

    AT+CSCS

**响应：**

::


    <CR><LF>OK<CR><LF>

^^^^

    短信服务模式选择

    下行/上行/广播支持
