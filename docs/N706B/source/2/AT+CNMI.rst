AT+CNMI：设置短信指示格式
-----------------

设置模组从网络接收到短信息后如何通知用户。
命令格式

命令格式
^^^^^^^^




**执行命令**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**命令：**

::

    AT+CNMI

**响应：**

::


    <CR><LF>OK<CR><LF>





参数
^^^^


- **<service>**：
  
    短信服务模式选择


- **<mt>,<mo>,<bm>**：
  
    下行/上行/广播支持
