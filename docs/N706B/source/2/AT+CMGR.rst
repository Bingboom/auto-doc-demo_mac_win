AT+CMGR：读短消息
-------------

读取当前存储器中的短消息（需预先通过AT+CPMS指令设定当前存储器）。
如果接收到的短信状态是未读的,执行该指令后,短信存储状态就变成已读。
命令格式

命令格式
^^^^^^^^




**执行命令**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**命令：**

::

    AT+CMGR

**响应：**

::

    <CR><LF>OK<CR><LF>





参数
^^^^


- **<service>**：
  短信服务模式选择


- **<mt>,<mo>,<bm>**：
  下行/上行/广播支持




说明
^^^^
说明示例36

示例命令
^^^^^^^^
::

    AT+CMGR
    
    OK
