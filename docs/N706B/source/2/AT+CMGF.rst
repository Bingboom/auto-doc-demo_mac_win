AT+CMGF：设置短消息模式
----------------

设置短信的输入模式。
命令格式

命令格式
^^^^^^^^




**执行命令**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**命令：**

::

    AT+CMGF

**响应：**

::

    <CR><LF>OK<CR><LF>





参数
^^^^


- **<service>**：
  短信服务模式选择


- **<mt>,<mo>,<bm>**：
  下行/上行/广播支持

^^^^
说明示例33

示例命令
^^^^^^^^
::

    AT+CMGF
