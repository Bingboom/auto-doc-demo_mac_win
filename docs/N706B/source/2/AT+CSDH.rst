AT+CSDH：显示文本模式参数
-----------------

设置是否在文本模式下的结果码中显示详细的头信息。该指令在短信文本模式下有效,需发送AT+CMGF=1设置成文本模式。
命令格式

命令格式
^^^^^^^^



**执行命令**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**命令：**

::

    AT+CSDH

**响应：**

::

    <CR><LF>OK<CR><LF>




参数
^^^^


- **<service>**： 短信服务模式选择


  - 0：GSM03.40/GSM03.41 Phase 2

  - 1：GSM03.40/GSM03.41 Phase 2+



- **<mt>,<mo>,<bm>**： 下行/上行/广播支持


  - 0：不支持

  - 1：支持
