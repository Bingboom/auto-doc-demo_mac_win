基础命令¶

  • ATI：获取模组厂商信息
  • AT+GMR：查询版本信息
  • AT+CSQ：获取信号强度
  • AT+CREG：查询网络注册状态
  • AT+CEREG：获取EPS网络注册状态
  • AT+COPS：网络选择
  • AT+CIMI：查询国际移动用户识别码
  • AT+CGSN：获取通信模组IMEI号
  • AT+GSN：获取通信模组IMEI号
  • AT+CCID：获取SIM卡标识
  • AT+CGMM：查询模组型号
  • AT+GMM：查询模组型号
  • AT+IPR：设置模组波特率
  • AT+CFUN：设置模组功能
  • AT+CMUX：串口多路复用指令
  • AT+CCLK：时钟管理
  • AT+CPIN：输入PIN码
  • AT+CLCK：PIN使能与查询功能指令
  • AT+CPWD：修改密码指令
  • AT+CGDCONT：设置PDP格式
  • AT+XGAUTH：用户认证
  • AT+CGATT：设置GPRS附着和分离
  • ATE1：/ATE0：打开&关闭回显
  • ATD：*99#：GPRS
  • AT+CESQ：扩展信号强度
  • AT+NWDNS：域名解析
  • AT+NWENPWRSAVE：休眠（Sleep）设置
  • AT+NWPWROFF：模组关机指令
  • AT+NWRFTEST：模组强发强收指令（仅可用于测试）
  • AT+NSTGETRSSI：获取强收模式下模组的接收功率（仅可用于测试）
