AT+CMGL：短信列表¶

读取某一类存储的短信,短信会被从+CPMS 指令选取的当前的存储器中读出来。 命令格式

命令格式¶

执行命令¶

命令：

AT+CMGL

响应：

<CR><LF>OK<CR><LF>

参数¶

  • <service>： 短信服务模式选择

  • 0：GSM03.40/GSM03.41 Phase 2

  • 1：GSM03.40/GSM03.41 Phase 2+

  • <mt>,<mo>,<bm>： 下行/上行/广播支持

  • 0：不支持

  • 1：支持

命令示例¶

AT+CMGL

OK
