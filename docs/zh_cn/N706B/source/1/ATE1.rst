ATE1：打开&关闭回显
----------------

打开（或关闭）模组AT指令回显功能。
该模组默认回显功能为打开状态。
该指令设置后掉电不保存。
命令格式

命令格式
^^^^^^^^







**执行命令**

命令：
::

    ATE1

响应：
::


    <CR><LF>OK<CR><LF>





参数
^^^^^^^^


- **<service>**： 短信服务模式选择

  - 0：GSM03.40/GSM03.41 Phase 2

  - 1：GSM03.40/GSM03.41 Phase 2+


- **<mt>,<mo>,<bm>**： 下行/上行/广播支持

  - 0：不支持

  - 1：支持





命令示例
^^^^^^^^


  
    
  



::

    ATE1
    
    OK
