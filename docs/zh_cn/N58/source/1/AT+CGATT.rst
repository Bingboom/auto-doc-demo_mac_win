AT+CGATT：设置GPRS附着和分离
------------------------

该指令用来查询、设置GPRS附着和分离。掉电不保存。
模组默认情况下,会主动进行GPRS附着。
进行PPP连接之前要确保GPRS是处于附着状态,AT流程增加查询指令AT+CGATT?：
如果返回值是1,则可以直接进行PPP连接,如果返回值是0,则需进行手动附着,即AT+CGATT=1。
命令格式

命令格式
^^^^^^^^







**执行命令**

命令：
::

    AT+CGATT

响应：
::


    <CR><LF>OK<CR><LF>





参数
^^^^^^^^


- **<service>**： 短信服务模式选择

  - 0：GSM03.40/GSM03.41 Phase 2

  - 1：GSM03.40/GSM03.41 Phase 2+


- **<mt>,<mo>,<bm>**： 下行/上行/广播支持

  - 0：不支持

  - 1：支持





命令示例
^^^^^^^^


  
    
  



::

    AT+CGATT
    
    OK
