AT+CCLK：时钟管理
----------------

设置和查询模组的实时时钟。
设置的时间立即生效,掉电保存,默认时钟为0时区,使用1/4时区。
命令格式

命令格式
^^^^^^^^








**执行命令**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**命令：**
::

    AT+CCLK

**响应：**
::


    <CR><LF>OK<CR><LF>




参数
^^^^^^^^


- **<service>**： 短信服务模式选择

  - 0：GSM03.40/GSM03.41 Phase 2

  - 1：GSM03.40/GSM03.41 Phase 2+


- **<mt>,<mo>,<bm>**： 下行/上行/广播支持

  - 0：不支持

  - 1：支持




命令示例
^^^^^^^^


  
    
  



::


    AT+CCLK
    
    OK
