AT+CPWD：修改密码指令
------------------

修改模组锁功能的密码。
若需修改PIN码,需锁定SIM卡（AT+CLCK="SC",1,"1234"）后才能修改。
命令格式

命令格式
^^^^^^^^








**执行命令**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**命令：**
::

    AT+CPWD

**响应：**
::


    <CR><LF>OK<CR><LF>




参数
^^^^^^^^


- **<service>**： 短信服务模式选择

  - 0：GSM03.40/GSM03.41 Phase 2

  - 1：GSM03.40/GSM03.41 Phase 2+


- **<mt>,<mo>,<bm>**： 下行/上行/广播支持

  - 0：不支持

  - 1：支持




命令示例
^^^^^^^^


  
    
  



::


    AT+CPWD
    
    OK
