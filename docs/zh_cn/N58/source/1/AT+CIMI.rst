AT+CIMI：查询国际移动用户识别码
-----------------------

获取国际移动用户识别码IMSI（international mobile subscriber identification）。
命令格式

命令格式
^^^^^^^^








**执行命令**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**命令：**
::

    AT+CIMI

**响应：**
::


    <CR><LF>OK<CR><LF>




参数
^^^^^^^^


- **<service>**： 短信服务模式选择

  - 0：GSM03.40/GSM03.41 Phase 2

  - 1：GSM03.40/GSM03.41 Phase 2+


- **<mt>,<mo>,<bm>**： 下行/上行/广播支持

  - 0：不支持

  - 1：支持




命令示例
^^^^^^^^


  
    
  



::


    AT+CIMI
    
    OK
