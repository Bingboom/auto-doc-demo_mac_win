


.. _cmd-at+cmgd:

AT+CMGD：删除短消息
-----------------

从当前存储器中删除短消息。
命令格式

命令格式
^^^^^^^^







**执行命令**

命令：
::

    AT+CMGD

响应：
::


    <CR><LF>OK<CR><LF>





参数
^^^^^^^^


- **<service>**： 短信服务模式选择

  - 0：GSM03.40/GSM03.41 Phase 2

  - 1：GSM03.40/GSM03.41 Phase 2+


- **<mt>,<mo>,<bm>**： 下行/上行/广播支持

  - 0：不支持

  - 1：支持





命令示例
^^^^^^^^


  
    
  



::

    AT+CMGD
    
    OK
