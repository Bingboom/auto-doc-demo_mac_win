AT+CMGW：写短消息
----------------

往存储器中写入短消息,正确存储后返回位置信息<index>。
命令格式

命令格式
^^^^^^^^








**执行命令**
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

**命令：**
::

    AT+CMGW

**响应：**
::


    <CR><LF>OK<CR><LF>




参数
^^^^^^^^


- **<service>**： 短信服务模式选择

  - 0：GSM03.40/GSM03.41 Phase 2

  - 1：GSM03.40/GSM03.41 Phase 2+


- **<mt>,<mo>,<bm>**： 下行/上行/广播支持

  - 0：不支持

  - 1：支持




命令示例
^^^^^^^^


  
    
  



::


    AT+CMGW
    
    OK
