SMS命令¶

  • AT+CSMS：选择短信服务
  • AT+CPMS：首选短信存储器
  • AT+CMGF：设置短消息模式
  • AT+CSCS：设置TE字符集
  • AT+CNMI：设置短信指示格式
  • AT+CMGR：读短消息
  • AT+CMGL：短信列表
  • AT+CMGS：发送短消息
  • AT+CMGW：写短消息
  • AT+CMSS：发送已保存的短消息
  • AT+CMGD：删除短消息
  • AT+CSCA：短信中心号码
  • AT+CSMP：设置文本模式参数
  • AT+CSDH：显示文本模式参数
